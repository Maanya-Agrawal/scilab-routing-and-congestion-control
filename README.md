# Scilab Routing and Congestion Control

## Description

This project demonstrates the implementation and analysis of routing and congestion control algorithms using Scilab. Network topologies of varying sizes are generated and analyzed using Bellman-Ford and Dijkstra algorithms. Congestion control and congestion mapping are also simulated to study network performance under different conditions.

## Objectives

* To implement routing using Bellman-Ford and Dijkstra algorithms
* To compare execution time for different network sizes
* To simulate congestion control in networks
* To analyze network performance using graphical representation

## Technologies Used

* Scilab

## Files Included

* `routing_analysis.sce`
  Implements routing algorithms and compares execution time

* `congestion_control.sce`
  Simulates congestion control for different network sizes

* `congestion_mapping.sce`
  Performs congestion mapping for varying node sizes

* `outputs/`
  Contains screenshots of network graphs, outputs, and performance plots

## How to Run

1. Open Scilab
2. Load any `.sce` file
3. Execute the script
4. Observe:

   * Network visualization
   * Execution time results
   * Performance graphs

## Conclusion

The experiment shows that execution time increases with network size. Dijkstra’s algorithm performs more efficiently than Bellman-Ford. Congestion control and mapping demonstrate how network load and size affect performance.
